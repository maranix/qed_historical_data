# qed_historical_data

A web scrapper written in GoLang to fetch Qutoes Equity Derivatives from NSE India website and exports them into an excel file. This project was made for my brother and might not suite for everyones use case.

#### Make sure you have go installed and running, if you don't then refer to the official [website](https://golang.org/).

## How to run
1. Clone ``` git clone https://github.com/ramanverma2k/qed_historical_data ```.
2. Change directory to the cloned repo.
3. Run ``` go get ```
4. Run ``` go run main.go ```
